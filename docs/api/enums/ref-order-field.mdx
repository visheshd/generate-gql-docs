---

Properties by which ref connections can be ordered.

```graphql
enum RefOrderField {
  ALPHABETICAL
  TAG_COMMIT_DATE
}
```

### Values

#### `ALPHABETICAL`

Order refs by their alphanumeric name

#### `TAG_COMMIT_DATE`

Order refs by underlying commit date if the ref prefix is refs/tags/
