The possible values for the IP allow list enabled setting.

```graphql
enum IpAllowListEnabledSettingValue {
  DISABLED
  ENABLED
}
```

### Values

#### `DISABLED`

The setting is disabled for the owner.

#### `ENABLED`

The setting is enabled for the owner.
