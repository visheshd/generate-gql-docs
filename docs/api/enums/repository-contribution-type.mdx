---

The reason a repository is listed as 'contributed'.

```graphql
enum RepositoryContributionType {
  COMMIT
  ISSUE
  PULL_REQUEST
  PULL_REQUEST_REVIEW
  REPOSITORY
}
```

### Values

#### `COMMIT`

Created a commit

#### `ISSUE`

Created an issue

#### `PULL_REQUEST`

Created a pull request

#### `PULL_REQUEST_REVIEW`

Reviewed a pull request

#### `REPOSITORY`

Created the repository
