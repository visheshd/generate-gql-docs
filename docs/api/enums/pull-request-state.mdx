---

The possible states of a pull request.

```graphql
enum PullRequestState {
  CLOSED
  MERGED
  OPEN
}
```

### Values

#### `CLOSED`

A pull request that has been closed without being merged.

#### `MERGED`

A pull request that has been closed by being merged.

#### `OPEN`

A pull request that is still open.
