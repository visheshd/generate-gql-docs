The possible team member roles; either 'maintainer' or 'member'.

```graphql
enum TeamMemberRole {
  MAINTAINER
  MEMBER
}
```

### Values

#### `MAINTAINER`

A team maintainer has permission to add and remove team members.

#### `MEMBER`

A team member has no administrative permissions on the team.
