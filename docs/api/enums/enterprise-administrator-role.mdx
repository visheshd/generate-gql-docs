The possible administrator roles in an enterprise account.

```graphql
enum EnterpriseAdministratorRole {
  BILLING_MANAGER
  OWNER
}
```

### Values

#### `BILLING_MANAGER`

Represents a billing manager of the enterprise account.

#### `OWNER`

Represents an owner of the enterprise account.
