---

Properties by which Sponsors tiers connections can be ordered.

```graphql
enum SponsorsTierOrderField {
  CREATED_AT
  MONTHLY_PRICE_IN_CENTS
}
```

### Values

#### `CREATED_AT`

Order tiers by creation time.

#### `MONTHLY_PRICE_IN_CENTS`

Order tiers by their monthly price in cents
