The possible roles for enterprise membership.

```graphql
enum EnterpriseUserAccountMembershipRole {
  MEMBER
  OWNER
}
```

### Values

#### `MEMBER`

The user is a member of the enterprise membership.

#### `OWNER`

The user is an owner of the enterprise membership.
