The billing plans available for organizations.

```graphql
enum OrgCreateAuditEntryBillingPlan {
  BUSINESS
  BUSINESS_PLUS
  FREE
  TIERED_PER_SEAT
  UNLIMITED
}
```

### Values

#### `BUSINESS`

Team Plan

#### `BUSINESS_PLUS`

Enterprise Cloud Plan

#### `FREE`

Free Plan

#### `TIERED_PER_SEAT`

Tiered Per Seat Plan

#### `UNLIMITED`

Legacy Unlimited Plan
