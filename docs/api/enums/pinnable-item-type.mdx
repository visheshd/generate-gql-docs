title: PinnableItemType
---

Represents items that can be pinned to a profile page or dashboard.

```graphql
enum PinnableItemType {
  GIST
  ISSUE
  ORGANIZATION
  PROJECT
  PULL_REQUEST
  REPOSITORY
  TEAM
  USER
}
```

### Values

#### `GIST`

A gist.

#### `ISSUE`

An issue.

#### `ORGANIZATION`

An organization.

#### `PROJECT`

A project.

#### `PULL_REQUEST`

A pull request.

#### `REPOSITORY`

A repository.

#### `TEAM`

A team.

#### `USER`

A user.
