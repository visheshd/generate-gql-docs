The permissions available to members on an Organization.

```graphql
enum OrgUpdateMemberAuditEntryPermission {
  ADMIN
  READ
}
```

### Values

#### `ADMIN`

Can read, clone, push, and add collaborators to repositories.

#### `READ`

Can read and clone repositories.
