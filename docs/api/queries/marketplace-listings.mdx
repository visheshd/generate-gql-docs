Look up Marketplace listings

```graphql
marketplaceListings(
  adminId: ID = 0
  after: String
  allStates: Boolean
  before: String
  categorySlug: String
  first: Int = 0
  last: Int = 0
  organizationId: ID = 0
  primaryCategoryOnly: Boolean
  slugs: [String] = []
  useTopicAliases: Boolean
  viewerCanAdmin: Boolean
  withFreeTrialsOnly: Boolean
): MarketplaceListingConnection!

```

### Arguments

#### `adminId` ([`ID`](/docs/api/scalars/id))

Select listings that can be administered by the specified user.

#### `after` ([`String`](/docs/api/scalars/string))

Returns the elements in the list that come after the specified cursor.

#### `allStates` ([`Boolean`](/docs/api/scalars/boolean))

Select listings visible to the viewer even if they are not approved. If omitted or
false, only approved listings will be returned.

#### `before` ([`String`](/docs/api/scalars/string))

Returns the elements in the list that come before the specified cursor.

#### `categorySlug` ([`String`](/docs/api/scalars/string))

Select only listings with the given category.

#### `first` ([`Int`](/docs/api/scalars/int))

Returns the first _n_ elements from the list.

#### `last` ([`Int`](/docs/api/scalars/int))

Returns the last _n_ elements from the list.

#### `organizationId` ([`ID`](/docs/api/scalars/id))

Select listings for products owned by the specified organization.

#### `primaryCategoryOnly` ([`Boolean`](/docs/api/scalars/boolean))

Select only listings where the primary category matches the given category slug.

#### `slugs` ([`[String]`](/docs/api/scalars/string))

Select the listings with these slugs, if they are visible to the viewer.

#### `useTopicAliases` ([`Boolean`](/docs/api/scalars/boolean))

Also check topic aliases for the category slug

#### `viewerCanAdmin` ([`Boolean`](/docs/api/scalars/boolean))

Select listings to which user has admin access. If omitted, listings visible to the
viewer are returned.

#### `withFreeTrialsOnly` ([`Boolean`](/docs/api/scalars/boolean))

Select only listings that offer a free trial.

### Type

#### [`MarketplaceListingConnection`](/docs/api/objects/marketplace-listing-connection)

Look up Marketplace Listings
