Look up a code of conduct by its key

```graphql
codeOfConduct(
  key: String!
): CodeOfConduct

```

### Arguments

#### `key` ([`String!`](/docs/api/scalars/string))

The code of conduct's key

### Type

#### [`CodeOfConduct`](/docs/api/objects/code-of-conduct)

The Code of Conduct for a repository
