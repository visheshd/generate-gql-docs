Represents an owner of a Repository.

```graphql
interface RepositoryOwner {
  avatarUrl(size: Int = 0): URI!
  id: ID!
  login: String!
  repositories(
    affiliations: [RepositoryAffiliation] = []
    after: String
    before: String
    first: Int = 0
    isFork: Boolean
    isLocked: Boolean
    last: Int = 0
    orderBy: RepositoryOrder
    ownerAffiliations: [RepositoryAffiliation] = [OWNER, COLLABORATOR]
    privacy: RepositoryPrivacy
  ): RepositoryConnection!
  repository(name: String!): Repository
  resourcePath: URI!
  url: URI!
}
```

### Fields

#### `avatarUrl` ([`URI!`](/docs/api/scalars/uri))

A URL pointing to the owner's public avatar.

- ##### `size` ([`Int`](/docs/api/scalars/int))

The size of the resulting square image.

#### `id` ([`ID!`](/docs/api/scalars/id))

#### `login` ([`String!`](/docs/api/scalars/string))

The username used to login.

#### `repositories` ([`RepositoryConnection!`](/docs/api/objects/repository-connection))

A list of repositories that the user owns.

- ##### `affiliations` ([`[RepositoryAffiliation]`](/docs/api/enums/repository-affiliation))

Array of viewer's affiliation options for repositories returned from the
connection. For example, OWNER will include only repositories that the
current viewer owns.

- ##### `after` ([`String`](/docs/api/scalars/string))

Returns the elements in the list that come after the specified cursor.

- ##### `before` ([`String`](/docs/api/scalars/string))

Returns the elements in the list that come before the specified cursor.

- ##### `first` ([`Int`](/docs/api/scalars/int))

Returns the first _n_ elements from the list.

- ##### `isFork` ([`Boolean`](/docs/api/scalars/boolean))

If non-null, filters repositories according to whether they are forks of another repository

- ##### `isLocked` ([`Boolean`](/docs/api/scalars/boolean))

If non-null, filters repositories according to whether they have been locked

- ##### `last` ([`Int`](/docs/api/scalars/int))

Returns the last _n_ elements from the list.

- ##### `orderBy` ([`RepositoryOrder`](/docs/api/inputs/repository-order))

Ordering options for repositories returned from the connection

- ##### `ownerAffiliations` ([`[RepositoryAffiliation]`](/docs/api/enums/repository-affiliation))

Array of owner's affiliation options for repositories returned from the
connection. For example, OWNER will include only repositories that the
organization or user being viewed owns.

- ##### `privacy` ([`RepositoryPrivacy`](/docs/api/enums/repository-privacy))

If non-null, filters repositories according to privacy

#### `repository` ([`Repository`](/docs/api/objects/repository))

Find Repository.

- ##### `name` ([`String!`](/docs/api/scalars/string))

Name of Repository to find.

#### `resourcePath` ([`URI!`](/docs/api/scalars/uri))

The HTTP URL for the owner.

#### `url` ([`URI!`](/docs/api/scalars/uri))

The HTTP URL for the owner.
