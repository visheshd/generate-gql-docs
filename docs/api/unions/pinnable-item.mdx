title: PinnableItem
---

Types that can be pinned to a profile page.

```graphql
union PinnableItem = Gist | Repository
```

### Possible types

#### [`Gist`](/docs/api/objects/gist)

A Gist.

#### [`Repository`](/docs/api/objects/repository)

A repository contains the content for a project.
