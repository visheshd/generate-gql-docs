Autogenerated input type of MergePullRequest

```graphql
type MergePullRequestInput {
  authorEmail: String
  clientMutationId: String

}
```

### Fields

#### `authorEmail` ([`String`](/docs/api/scalars/string))

The email address to associate with this merge.

#### `clientMutationId` ([`String`](/docs/api/scalars/string))

A unique identifier for the client performing the mutation.

#### `commitBody` ([`String`](/docs/api/scalars/string))

Commit body to use for the merge commit; if omitted, a default message will be used

#### `commitHeadline` ([`String`](/docs/api/scalars/string))

Commit headline to use for the merge commit; if omitted, a default message will be used.

#### `expectedHeadOid` ([`GitObjectID`](/docs/api/scalars/git-object-id))

OID that the pull request head ref must match to allow merge; if omitted, no check is performed.

#### `mergeMethod` ([`PullRequestMergeMethod`](/docs/api/enums/pull-request-merge-method))

The merge method to use. If omitted, defaults to 'MERGE'

#### `pullRequestId` ([`ID!`](/docs/api/scalars/id))

ID of the pull request to be merged.
