type MergeBranchInput {

### Fields

#### `authorEmail` ([`String`](/docs/api/scalars/string))

The email address to associate with this commit.

#### `base` ([`String!`](/docs/api/scalars/string))

The name of the base branch that the provided head will be merged into.

#### `clientMutationId` ([`String`](/docs/api/scalars/string))

A unique identifier for the client performing the mutation.

#### `commitMessage` ([`String`](/docs/api/scalars/string))

Message to use for the merge commit. If omitted, a default will be used.

#### `head` ([`String!`](/docs/api/scalars/string))

The head to merge into the base branch. This can be a branch name or a commit GitObjectID.

#### `repositoryId` ([`ID!`](/docs/api/scalars/id))

The Node ID of the Repository containing the base branch that will be modified.
