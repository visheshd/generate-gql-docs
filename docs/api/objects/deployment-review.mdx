A deployment review.

```graphql
type DeploymentReview implements Node {
  comment: String!
  databaseId: Int
  environments(
    after: String
    before: String
    first: Int = 0
    last: Int = 0

  state: DeploymentReviewState!
```

#### `id` ([`ID!`](/docs/api/scalars/id))
