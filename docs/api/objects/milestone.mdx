Represents a Milestone object on a given repository.

```graphql
type Milestone implements Closable, Node, UniformResourceLocatable {
  closed: Boolean!
  closedAt: DateTime
  createdAt: DateTime!
  creator: Actor
  description: String
  dueOn: DateTime
  id: ID!
  issues(
    after: String
    before: String
    filterBy: IssueFilters
    first: Int = 0
    labels: [String!] = []
    last: Int = 0
    orderBy: IssueOrder
    states: [IssueState!] = []
  ): IssueConnection!
  number: Int!
  progressPercentage: Float!
  pullRequests(
    after: String
    baseRefName: String
    before: String
    first: Int = 0
    headRefName: String
    labels: [String!] = []
    last: Int = 0
    orderBy: IssueOrder
    states: [PullRequestState!] = []
  ): PullRequestConnection!
  repository: Repository!
  resourcePath: URI!
  state: MilestoneState!
  title: String!
  updatedAt: DateTime!
  url: URI!
}
```

### Fields

#### `closed` ([`Boolean!`](/docs/api/scalars/boolean))

`true` if the object is closed (definition of closed may depend on type)

#### `closedAt` ([`DateTime`](/docs/api/scalars/date-time))

Identifies the date and time when the object was closed.

#### `createdAt` ([`DateTime!`](/docs/api/scalars/date-time))

Identifies the date and time when the object was created.

#### `creator` ([`Actor`](/docs/api/interfaces/actor))

Identifies the actor who created the milestone.

#### `description` ([`String`](/docs/api/scalars/string))

Identifies the description of the milestone.

#### `dueOn` ([`DateTime`](/docs/api/scalars/date-time))

Identifies the due date of the milestone.

#### `id` ([`ID!`](/docs/api/scalars/id))

#### `issues` ([`IssueConnection!`](/docs/api/objects/issue-connection))
