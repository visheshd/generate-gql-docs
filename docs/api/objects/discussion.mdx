A discussion in a repository.

```graphql
type Discussion implements Comment, Deletable, Labelable, Lockable, Node, Reactable, RepositoryNode, Subscribable, Updatable, Votable {
  activeLockReason: LockReason
  answer: DiscussionComment
  answerChosenAt: DateTime
  answerChosenBy: Actor
  author: Actor
  authorAssociation: CommentAuthorAssociation!
  body: String!
  bodyHTML: HTML!
  bodyText: String!
  category: DiscussionCategory!
  comments(
    after: String
    before: String
    first: Int = 0
    last: Int = 0
  ): DiscussionCommentConnection!
  createdAt: DateTime!
  createdViaEmail: Boolean!
  databaseId: Int
  editor: Actor
  id: ID!
  includesCreatedEdit: Boolean!
  labels(
    after: String
    before: String
    first: Int = 0
    last: Int = 0
    orderBy: LabelOrder = "[object Object]"
  ): LabelConnection
  lastEditedAt: DateTime
  locked: Boolean!
  number: Int!
  publishedAt: DateTime
  reactionGroups: [ReactionGroup!]
  reactions(
    after: String
    before: String
    content: ReactionContent
    first: Int = 0
    last: Int = 0
    orderBy: ReactionOrder
  ): ReactionConnection!
  repository: Repository!
  resourcePath: URI!
  title: String!
  updatedAt: DateTime!
  upvoteCount: Int!
  url: URI!
  userContentEdits(
    after: String
    before: String
    first: Int = 0
    last: Int = 0
  ): UserContentEditConnection
  viewerCanDelete: Boolean!
  viewerCanReact: Boolean!
  viewerCanSubscribe: Boolean!
  viewerCanUpdate: Boolean!
  viewerCanUpvote: Boolean!
  viewerDidAuthor: Boolean!
  viewerHasUpvoted: Boolean!
  viewerSubscription: SubscriptionState
}
```

### Fields

#### `activeLockReason` ([`LockReason`](/docs/api/enums/lock-reason))

Reason that the conversation was locked.

#### `answer` ([`DiscussionComment`](/docs/api/objects/discussion-comment))

The comment chosen as this discussion's answer, if any.

#### `answerChosenAt` ([`DateTime`](/docs/api/scalars/date-time))

The time when a user chose this discussion's answer, if answered.

#### `answerChosenBy` ([`Actor`](/docs/api/interfaces/actor))

The user who chose this discussion's answer, if answered.

#### `author` ([`Actor`](/docs/api/interfaces/actor))

The actor who authored the comment.

#### `authorAssociation` ([`CommentAuthorAssociation!`](/docs/api/enums/comment-author-association))

Author's association with the subject of the comment.

#### `body` ([`String!`](/docs/api/scalars/string))

The main text of the discussion post.

#### `bodyHTML` ([`HTML!`](/docs/api/scalars/html))

The body rendered to HTML.

#### `bodyText` ([`String!`](/docs/api/scalars/string))

The body rendered to text.

#### `category` ([`DiscussionCategory!`](/docs/api/objects/discussion-category))
