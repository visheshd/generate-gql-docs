A repository issue template.

```graphql
type IssueTemplate {
  about: String
  body: String
  name: String!
  title: String
}
```

### Fields

#### `about` ([`String`](/docs/api/scalars/string))

The template purpose.

#### `body` ([`String`](/docs/api/scalars/string))

The suggested issue body.

#### `name` ([`String!`](/docs/api/scalars/string))

The template name.

#### `title` ([`String`](/docs/api/scalars/string))

The suggested issue title.
