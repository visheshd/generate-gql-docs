SCIM attributes for the External Identity

```graphql
type ExternalIdentityScimAttributes {
  emails: [UserEmailMetadata!]
  familyName: String
  givenName: String
  groups: [String!]
  username: String
}
```

### Fields

#### `emails` ([`[UserEmailMetadata!]`](/docs/api/objects/user-email-metadata))

The emails associated with the SCIM identity

#### `familyName` ([`String`](/docs/api/scalars/string))

Family name of the SCIM identity

#### `givenName` ([`String`](/docs/api/scalars/string))

Given name of the SCIM identity

#### `groups` ([`[String!]`](/docs/api/scalars/string))

The groups linked to this identity in IDP

#### `username` ([`String`](/docs/api/scalars/string))

The userName of the SCIM identity
