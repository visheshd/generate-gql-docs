A special type of user which takes actions on behalf of GitHub Apps.

```graphql
type Bot implements Actor, Node, UniformResourceLocatable {
  avatarUrl(size: Int = 0): URI!
  createdAt: DateTime!
  databaseId: Int
  id: ID!
  login: String!
  resourcePath: URI!
  updatedAt: DateTime!
  url: URI!
}
```

### Fields

#### `avatarUrl` ([`URI!`](/docs/api/scalars/uri))

A URL pointing to the GitHub App's public avatar.

- ##### `size` ([`Int`](/docs/api/scalars/int))

The size of the resulting square image.

#### `createdAt` ([`DateTime!`](/docs/api/scalars/date-time))

Identifies the date and time when the object was created.

#### `databaseId` ([`Int`](/docs/api/scalars/int))

Identifies the primary key from the database.

#### `id` ([`ID!`](/docs/api/scalars/id))

#### `login` ([`String!`](/docs/api/scalars/string))

The username of the actor.

#### `resourcePath` ([`URI!`](/docs/api/scalars/uri))

The HTTP path for this bot

#### `updatedAt` ([`DateTime!`](/docs/api/scalars/date-time))

Identifies the date and time when the object was last updated.

#### `url` ([`URI!`](/docs/api/scalars/uri))

The HTTP URL for this bot

### Interfaces

#### [`Actor`](/docs/api/interfaces/actor)

Represents an object which can take actions on GitHub. Typically a User or Bot.

#### [`Node`](/docs/api/interfaces/node)

An object with an ID.

#### [`UniformResourceLocatable`](/docs/api/interfaces/uniform-resource-locatable)

Represents a type that can be retrieved by a URL.
