title: UnknownSignature
---

Represents an unknown signature on a Commit or Tag.

```graphql
type UnknownSignature implements GitSignature {
  email: String!
  isValid: Boolean!
  payload: String!
  signature: String!
  signer: User
  state: GitSignatureState!
  wasSignedByGitHub: Boolean!
}
```

### Fields

#### `email` ([`String!`](/docs/api/scalars/string))

Email used to sign this object.

#### `isValid` ([`Boolean!`](/docs/api/scalars/boolean))

True if the signature is valid and verified by GitHub.

#### `payload` ([`String!`](/docs/api/scalars/string))

Payload for GPG signing object. Raw ODB object without the signature header.

#### `signature` ([`String!`](/docs/api/scalars/string))

ASCII-armored signature header from object.

#### `signer` ([`User`](/docs/api/objects/user))

GitHub user corresponding to the email signing this commit.

#### `state` ([`GitSignatureState!`](/docs/api/enums/git-signature-state))

The state of this signature. `VALID` if signature is valid and verified by
GitHub, otherwise represents reason why signature is considered invalid.

#### `wasSignedByGitHub` ([`Boolean!`](/docs/api/scalars/boolean))

True if the signature was made with GitHub's signing key.

### Interfaces

#### [`GitSignature`](/docs/api/interfaces/git-signature)

Information about a signature (GPG or S/MIME) on a Commit or Tag.
