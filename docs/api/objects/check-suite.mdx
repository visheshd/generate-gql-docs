A check suite.

```graphql
type CheckSuite implements Node {
  app: App
  branch: Ref
  checkRuns(
    after: String
    before: String
    filterBy: CheckRunFilter
    first: Int = 0
    last: Int = 0
  ): CheckRunConnection
  commit: Commit!
  conclusion: CheckConclusionState
  createdAt: DateTime!
  creator: User
  databaseId: Int
  id: ID!
  matchingPullRequests(
    after: String
    baseRefName: String
    before: String
    first: Int = 0
    headRefName: String
    labels: [String!] = []
    last: Int = 0
    orderBy: IssueOrder
    states: [PullRequestState!] = []
  ): PullRequestConnection
  push: Push
  repository: Repository!
  resourcePath: URI!
  status: CheckStatusState!
  updatedAt: DateTime!
  url: URI!
  workflowRun: WorkflowRun
}
```

### Fields

#### `app` ([`App`](/docs/api/objects/app))

The GitHub App which created this check suite.

#### `branch` ([`Ref`](/docs/api/objects/ref))

The name of the branch for this check suite.

#### `checkRuns` ([`CheckRunConnection`](/docs/api/objects/check-run-connection))

The check runs associated with a check suite.

- ##### `after` ([`String`](/docs/api/scalars/string))

- ##### `before` ([`String`](/docs/api/scalars/string))

Returns the elements in the list that come before the specified cursor.

- ##### `filterBy` ([`CheckRunFilter`](/docs/api/inputs/check-run-filter))

Filters the check runs by this type.

- ##### `first` ([`Int`](/docs/api/scalars/int))

Returns the first _n_ elements from the list.

- ##### `last` ([`Int`](/docs/api/scalars/int))

Returns the last _n_ elements from the list.

#### `commit` ([`Commit!`](/docs/api/objects/commit))

The commit for this check suite

#### `conclusion` ([`CheckConclusionState`](/docs/api/enums/check-conclusion-state))

The conclusion of this check suite.

#### `createdAt` ([`DateTime!`](/docs/api/scalars/date-time))

Identifies the date and time when the object was created.

#### `creator` ([`User`](/docs/api/objects/user))

The user who triggered the check suite.

#### `databaseId` ([`Int`](/docs/api/scalars/int))

#### `id` ([`ID!`](/docs/api/scalars/id))

#### `matchingPullRequests` ([`PullRequestConnection`](/docs/api/objects/pull-request-connection))

- ##### `after` ([`String`](/docs/api/scalars/string))

Returns the elements in the list that come after the specified cursor.

- ##### `baseRefName` ([`String`](/docs/api/scalars/string))

The base ref name to filter the pull requests by.

- ##### `before` ([`String`](/docs/api/scalars/string))

Returns the elements in the list that come before the specified cursor.

- ##### `first` ([`Int`](/docs/api/scalars/int))

Returns the first _n_ elements from the list.

- ##### `headRefName` ([`String`](/docs/api/scalars/string))

- ##### `labels` ([`[String!]`](/docs/api/scalars/string))

#### `push` ([`Push`](/docs/api/objects/push))
