A comment on a team discussion.

```graphql
type TeamDiscussionComment implements Comment, Deletable, Node, Reactable, UniformResourceLocatable, Updatable, UpdatableComment {
  author: Actor
  authorAssociation: CommentAuthorAssociation!
  body: String!
  bodyHTML: HTML!
  bodyText: String!
  bodyVersion: String!
  createdAt: DateTime!
  createdViaEmail: Boolean!
  databaseId: Int
  discussion: TeamDiscussion!
  editor: Actor
  id: ID!
  includesCreatedEdit: Boolean!
  lastEditedAt: DateTime
  number: Int!
  publishedAt: DateTime
  reactionGroups: [ReactionGroup!]
  reactions(
    after: String
    before: String
    content: ReactionContent
    first: Int = 0
    last: Int = 0
    orderBy: ReactionOrder
  ): ReactionConnection!
  resourcePath: URI!
  updatedAt: DateTime!
  url: URI!
  userContentEdits(
    after: String
    before: String
    first: Int = 0
    last: Int = 0
  ): UserContentEditConnection
  viewerCanDelete: Boolean!
  viewerCanReact: Boolean!
  viewerCanUpdate: Boolean!
  viewerCannotUpdateReasons: [CommentCannotUpdateReason!]!
  viewerDidAuthor: Boolean!
}
```

### Fields

#### `author` ([`Actor`](/docs/api/interfaces/actor))

The actor who authored the comment.

#### `authorAssociation` ([`CommentAuthorAssociation!`](/docs/api/enums/comment-author-association))

Author's association with the comment's team.

#### `body` ([`String!`](/docs/api/scalars/string))

The body as Markdown.

#### `bodyHTML` ([`HTML!`](/docs/api/scalars/html))

The body rendered to HTML.

#### `bodyText` ([`String!`](/docs/api/scalars/string))

The body rendered to text.

#### `bodyVersion` ([`String!`](/docs/api/scalars/string))

The current version of the body content.

#### `createdAt` ([`DateTime!`](/docs/api/scalars/date-time))

Identifies the date and time when the object was created.

#### `createdViaEmail` ([`Boolean!`](/docs/api/scalars/boolean))

Check if this comment was created via an email reply.

#### `databaseId` ([`Int`](/docs/api/scalars/int))

Identifies the primary key from the database.

#### `discussion` ([`TeamDiscussion!`](/docs/api/objects/team-discussion))

The discussion this comment is about.

#### `editor` ([`Actor`](/docs/api/interfaces/actor))

The actor who edited the comment.

#### `id` ([`ID!`](/docs/api/scalars/id))

#### `includesCreatedEdit` ([`Boolean!`](/docs/api/scalars/boolean))

Check if this comment was edited and includes an edit with the creation data

#### `lastEditedAt` ([`DateTime`](/docs/api/scalars/date-time))

The moment the editor made the last edit

#### `number` ([`Int!`](/docs/api/scalars/int))

Identifies the comment number.

#### `publishedAt` ([`DateTime`](/docs/api/scalars/date-time))

Identifies when the comment was published at.

#### `reactionGroups` ([`[ReactionGroup!]`](/docs/api/objects/reaction-group))

A list of reactions grouped by content left on the subject.

#### `reactions` ([`ReactionConnection!`](/docs/api/objects/reaction-connection))

A list of Reactions left on the Issue.

- ##### `after` ([`String`](/docs/api/scalars/string))

Returns the elements in the list that come after the specified cursor.

- ##### `before` ([`String`](/docs/api/scalars/string))

Returns the elements in the list that come before the specified cursor.

- ##### `content` ([`ReactionContent`](/docs/api/enums/reaction-content))

Allows filtering Reactions by emoji.

- ##### `first` ([`Int`](/docs/api/scalars/int))

Returns the first _n_ elements from the list.

- ##### `last` ([`Int`](/docs/api/scalars/int))

Returns the last _n_ elements from the list.

- ##### `orderBy` ([`ReactionOrder`](/docs/api/inputs/reaction-order))

Allows specifying the order in which reactions are returned.

#### `resourcePath` ([`URI!`](/docs/api/scalars/uri))

The HTTP path for this comment

#### `updatedAt` ([`DateTime!`](/docs/api/scalars/date-time))

Identifies the date and time when the object was last updated.

#### `url` ([`URI!`](/docs/api/scalars/uri))

The HTTP URL for this comment

#### `userContentEdits` ([`UserContentEditConnection`](/docs/api/objects/user-content-edit-connection))

A list of edits to this content.

- ##### `after` ([`String`](/docs/api/scalars/string))

Returns the elements in the list that come after the specified cursor.

- ##### `before` ([`String`](/docs/api/scalars/string))

Returns the elements in the list that come before the specified cursor.

- ##### `first` ([`Int`](/docs/api/scalars/int))

Returns the first _n_ elements from the list.

- ##### `last` ([`Int`](/docs/api/scalars/int))

Returns the last _n_ elements from the list.

#### `viewerCanDelete` ([`Boolean!`](/docs/api/scalars/boolean))

Check if the current viewer can delete this object.

#### `viewerCanReact` ([`Boolean!`](/docs/api/scalars/boolean))

Can user react to this subject

#### `viewerCanUpdate` ([`Boolean!`](/docs/api/scalars/boolean))

Check if the current viewer can update this object.

#### `viewerCannotUpdateReasons` ([`[CommentCannotUpdateReason!]!`](/docs/api/enums/comment-cannot-update-reason))

Reasons why the current viewer can not update this comment.

#### `viewerDidAuthor` ([`Boolean!`](/docs/api/scalars/boolean))

Did the viewer author this comment.
