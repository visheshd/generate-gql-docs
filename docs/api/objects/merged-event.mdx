title: MergedEvent
---

Represents a 'merged' event on a given pull request.

```graphql
type MergedEvent implements Node, UniformResourceLocatable {
  actor: Actor
  commit: Commit
  createdAt: DateTime!
  id: ID!
  mergeRef: Ref
  mergeRefName: String!
  pullRequest: PullRequest!
  resourcePath: URI!
  url: URI!
}
```

### Fields

#### `actor` ([`Actor`](/docs/api/interfaces/actor))

Identifies the actor who performed the event.

#### `commit` ([`Commit`](/docs/api/objects/commit))

Identifies the commit associated with the `merge` event.

#### `createdAt` ([`DateTime!`](/docs/api/scalars/date-time))

Identifies the date and time when the object was created.

#### `id` ([`ID!`](/docs/api/scalars/id))

#### `mergeRef` ([`Ref`](/docs/api/objects/ref))

Identifies the Ref associated with the `merge` event.

#### `mergeRefName` ([`String!`](/docs/api/scalars/string))

Identifies the name of the Ref associated with the `merge` event.

#### `pullRequest` ([`PullRequest!`](/docs/api/objects/pull-request))

PullRequest referenced by event.

#### `resourcePath` ([`URI!`](/docs/api/scalars/uri))

The HTTP path for this merged event.

#### `url` ([`URI!`](/docs/api/scalars/uri))

The HTTP URL for this merged event.

### Interfaces

#### [`Node`](/docs/api/interfaces/node)

An object with an ID.

#### [`UniformResourceLocatable`](/docs/api/interfaces/uniform-resource-locatable)

Represents a type that can be retrieved by a URL.
