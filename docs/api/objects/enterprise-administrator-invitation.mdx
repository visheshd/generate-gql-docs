title: EnterpriseAdministratorInvitation
---

An invitation for a user to become an owner or billing manager of an enterprise.

```graphql
type EnterpriseAdministratorInvitation implements Node {
  createdAt: DateTime!
  email: String
  enterprise: Enterprise!
  id: ID!
  invitee: User
  inviter: User
  role: EnterpriseAdministratorRole!
}
```

### Fields

#### `createdAt` ([`DateTime!`](/docs/api/scalars/date-time))

Identifies the date and time when the object was created.

#### `email` ([`String`](/docs/api/scalars/string))

The email of the person who was invited to the enterprise.

#### `enterprise` ([`Enterprise!`](/docs/api/objects/enterprise))

The enterprise the invitation is for.

#### `id` ([`ID!`](/docs/api/scalars/id))

#### `invitee` ([`User`](/docs/api/objects/user))

The user who was invited to the enterprise.

#### `inviter` ([`User`](/docs/api/objects/user))

The user who created the invitation.

#### `role` ([`EnterpriseAdministratorRole!`](/docs/api/enums/enterprise-administrator-role))

The invitee's pending role in the enterprise (owner or billing_manager).

### Interfaces

#### [`Node`](/docs/api/interfaces/node)

An object with an ID.
