title: ReviewRequest
---

A request for a user to review a pull request.

```graphql
type ReviewRequest implements Node {
  asCodeOwner: Boolean!
  databaseId: Int
  id: ID!
  pullRequest: PullRequest!
  requestedReviewer: RequestedReviewer
}
```

### Fields

#### `asCodeOwner` ([`Boolean!`](/docs/api/scalars/boolean))

Whether this request was created for a code owner

#### `databaseId` ([`Int`](/docs/api/scalars/int))

Identifies the primary key from the database.

#### `id` ([`ID!`](/docs/api/scalars/id))

#### `pullRequest` ([`PullRequest!`](/docs/api/objects/pull-request))

Identifies the pull request associated with this review request.

#### `requestedReviewer` ([`RequestedReviewer`](/docs/api/unions/requested-reviewer))

The reviewer that is requested.

### Interfaces

#### [`Node`](/docs/api/interfaces/node)

An object with an ID.
