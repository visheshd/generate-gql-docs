A topic aggregates entities that are related to a subject.

```graphql
type Topic implements Node, Starrable {
  id: ID!
  name: String!
  relatedTopics(first: Int = 3): [Topic!]!
  stargazerCount: Int!
  stargazers(
    after: String
    before: String
    first: Int = 0
    last: Int = 0
    orderBy: StarOrder
  ): StargazerConnection!
  viewerHasStarred: Boolean!
}
```

### Fields

#### `id` ([`ID!`](/docs/api/scalars/id))

#### `name` ([`String!`](/docs/api/scalars/string))

The topic's name.

#### `relatedTopics` ([`[Topic!]!`](/docs/api/objects/topic))

A list of related topics, including aliases of this topic, sorted with the most relevant
first. Returns up to 10 Topics.

- ##### `first` ([`Int`](/docs/api/scalars/int))

How many topics to return.

#### `stargazerCount` ([`Int!`](/docs/api/scalars/int))

Returns a count of how many stargazers there are on this object

#### `stargazers` ([`StargazerConnection!`](/docs/api/objects/stargazer-connection))

A list of users who have starred this starrable.

- ##### `after` ([`String`](/docs/api/scalars/string))

Returns the elements in the list that come after the specified cursor.

- ##### `before` ([`String`](/docs/api/scalars/string))

Returns the elements in the list that come before the specified cursor.

- ##### `first` ([`Int`](/docs/api/scalars/int))

Returns the first _n_ elements from the list.

- ##### `last` ([`Int`](/docs/api/scalars/int))

Returns the last _n_ elements from the list.

- ##### `orderBy` ([`StarOrder`](/docs/api/inputs/star-order))

Order for connection

#### `viewerHasStarred` ([`Boolean!`](/docs/api/scalars/boolean))

Returns a boolean indicating whether the viewing user has starred this starrable.

### Interfaces

#### [`Node`](/docs/api/interfaces/node)

An object with an ID.

#### [`Starrable`](/docs/api/interfaces/starrable)

Things that can be starred.
