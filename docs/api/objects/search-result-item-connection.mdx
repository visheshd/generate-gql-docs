A list of results that matched against a search query.

```graphql
type SearchResultItemConnection {
  codeCount: Int!
  discussionCount: Int!
  edges: [SearchResultItemEdge]
  issueCount: Int!
  nodes: [SearchResultItem]
  pageInfo: PageInfo!
  repositoryCount: Int!
  userCount: Int!
  wikiCount: Int!
}
```

### Fields

#### `codeCount` ([`Int!`](/docs/api/scalars/int))

The number of pieces of code that matched the search query.

#### `discussionCount` ([`Int!`](/docs/api/scalars/int))

The number of discussions that matched the search query.

#### `edges` ([`[SearchResultItemEdge]`](/docs/api/objects/search-result-item-edge))

A list of edges.

#### `issueCount` ([`Int!`](/docs/api/scalars/int))

The number of issues that matched the search query.

#### `nodes` ([`[SearchResultItem]`](/docs/api/unions/search-result-item))

A list of nodes.

#### `pageInfo` ([`PageInfo!`](/docs/api/objects/page-info))

Information to aid in pagination.

#### `repositoryCount` ([`Int!`](/docs/api/scalars/int))

The number of repositories that matched the search query.

#### `userCount` ([`Int!`](/docs/api/scalars/int))

The number of users that matched the search query.

#### `wikiCount` ([`Int!`](/docs/api/scalars/int))

The number of wiki pages that matched the search query.
