title: GitActor
---

Represents an actor in a Git commit (ie. an author or committer).

```graphql
type GitActor {
  avatarUrl(size: Int = 0): URI!
  date: GitTimestamp
  email: String
  name: String
  user: User
}
```

### Fields

#### `avatarUrl` ([`URI!`](/docs/api/scalars/uri))

A URL pointing to the author's public avatar.

- ##### `size` ([`Int`](/docs/api/scalars/int))

The size of the resulting square image.

#### `date` ([`GitTimestamp`](/docs/api/scalars/git-timestamp))

The timestamp of the Git action (authoring or committing).

#### `email` ([`String`](/docs/api/scalars/string))

The email in the Git commit.

#### `name` ([`String`](/docs/api/scalars/string))

The name in the Git commit.

#### `user` ([`User`](/docs/api/objects/user))

The GitHub user corresponding to the email field. Null if no such user exists.
